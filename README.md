## 🧾 Overview  
<a id="overview"></a>  
This mini project is a console-based game inspired by the classic **Snake, Water, Gun** game (similar to Rock-Paper-Scissors).  

In this game:
- Snake drinks water  
- Water damages gun  
- Gun kills snake  

The user plays against the computer, and the winner is decided based on predefined rules.

This project helps beginners understand:
- Random number generation  
- Conditional logic  
- User input handling  
- Game development basics  

---

## 🏢 Problem Statement (Business Problem)  
<a id="problem-statement"></a>  
Interactive games are widely used in entertainment and learning platforms to improve engagement.  

This project simulates a simple game system where:
- A user competes against the computer  
- The system generates random choices  
- The winner is determined using logic rules

## 🔧 Method  
<a id="method"></a>  
1. Import the `random` module  
2. Define choices (Snake, Water, Gun)  
3. Take user input  
4. Generate computer choice randomly  
5. Compare both choices using conditions  
6. Display the winner
